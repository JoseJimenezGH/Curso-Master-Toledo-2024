Caso de estudio: Golondrina común Hirundo rustica

Vamos a considerar el caso más simple de una matriz de proyección o transición.

Definimos los parametros vitales que utilizaremos -combinados- para crear la matriz de población:

\[ \begin{align*} &f_{1}= 1.50 && \text{ # Fecundidad de las hembras juveniles (de primer año)}\\ &f_{2}= 3.00 && \text{ # Fecundidad de las hembras adultas (de más de un año)}\\ &s_{0}= 0.20 && \text{ # Supervivencia de los pollos (edad: < 1)}\\ &s_{1}= 0.50 && \text{ # Supervivencia juvenil (edad: 1)}\\ &s_{2}= 0.65 && \text{ # Supervivencia adulta (edad: > 1)}\\ \end{align*} \] Inmediatamente antes de la reproducción, la productividad y la supervivencia se pueden expresar en función de estos parámetros, y así, los pollos en un año t+1 va a depender de la supervivencia y la productividad de las hembras de primer año y adultas en el año t, y la población reproductora en ese año t+1 va a depender de la supervivencia de las hembras de primer año y adultas en el año t:

\[ \begin{aligned} N_{1,t+1}=& s_{0}*f_{1}*N_{1,t}+s_{0}*f_{2}*N_{2,t} && \text{ # Número de pollos producidos}\\ N_{2,t+1}=& s_{1}*N_{1,t}+s_{2}*N_{2,t} && \text{ # Número de juveniles y adultos en la población}\\ \end{aligned} \]

Eso, en forma matricial, es:

\[ \begin{bmatrix} N_{1}\\N_{2}\\ \end{bmatrix} = \begin{bmatrix}s_{0}*f_{1} & s_{0}*f_{2}\\ s_{1} & s_{2} \end{bmatrix}*\begin{bmatrix}N_{1}\\N_{2}\end{bmatrix}\\ \]

Podemos sustituir por los valores:

\[ \begin{bmatrix} N_{1}\\N_{2}\\ \end{bmatrix} = \begin{bmatrix}0.30 & 0.60\\ 0.50 & 0.65 \end{bmatrix}*\begin{bmatrix}N_{1}\\N_{2}\end{bmatrix}\\ \] Vamos a hacer una proyección en R usando esa matriz A:

f1 <- 1.5   # Fecundidad de edad clase 1 (hembras de primer año)
f2 <- 3     # Fecundidad de edad clase 2 (hembras de 2 años o mas)
s0 <- 0.2   # Supervivencia de los pollos
s1 <- 0.5   # Supervivencia juvenil
s2 <- 0.65  # Supervivencia adulta

A <- matrix(c(
  s0*f1, s0*f2,
  s1,    s2), nrow = 2, byrow = TRUE)

tspan <- 20            # período de tiempo
rows <- dim(A)[1]

Creamos unas matrices para contener los resultados

n <- matrix(0,rows,tspan)     # abundancia, por clases de edad
N <- matrix(0,tspan,1)        # abundancia total
gr <- matrix(0,tspan-1,1)     # tasa de crecimiento, por años

Población inicial por clases de edad:

n[,1] <- c(10,0)

Resultados

for (t in 1:(tspan-1)) {
  n[,t+1] <- A%*%n[,t]              # %*% = multiplicación del matrices en R
  N[t+1] <- sum(n[,t+1])
  gr[t] <- sum(n[,t+1])/sum(n[,t])  # tasa de crecimiento
}

Vamos a calcular sensibilidad y elasticidad

library(popbio)
gol.vr <- list(s0=0.2, s1=0.5, s2=0.65,f1=1.5, f2=3)
el <- expression(s0*f1, s0*f2,
                 s1,    s2)

Veamos como el efecto en los cambios en los ratios vitales afecta a la tasa de crecimiento. Con el análisis de sensibilidad se evaluan los cambios absolutos en la tasa de crecimiento poblacional (\(\lambda\)), y con el análisis de elasticidad la contribución porcentual de cada ratio vital.

n <- length(gol.vr)
vr <- seq(0.1,1,.1)

vrsen <- matrix(numeric(n*length(vr)), ncol=n, dimnames=list(vr, names(gol.vr)))
for (h in 1:n) {
   gol.vr2 <- list(s0=0.2, s1=0.5, s2=0.65,f1=1.5, f2=3)
   for (i in 1:length(vr)){
      gol.vr2[[h]] <- vr[i]
      A <- matrix(sapply(el, eval,gol.vr2 , NULL), nrow=sqrt(length(el)), byrow=TRUE)
      vrsen[i,h] <- max(Re(eigen(A)$values))
   }
}
matplot(rownames(vrsen), vrsen, type='l', lwd=2, las=1,
ylab="Population population growth", xlab="Value of vital rate",
main="Effects of changing  vital rates")
